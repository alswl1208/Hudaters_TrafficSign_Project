# Traffic Sign Change Detection Project

4-2 산학프로젝트

□ 프로젝트 과제 : 물류 차량 영상을 이용한 도로 변화 정보 탐지
- 참여기간 : 2025년 3월 ~ 2025년 12월

## 개요

본 프로젝트는 물류 차량의 블랙박스 영상을 인공지능으로 분석하여 도로 표지판의 변화를 자동 탐지하고, 이를 실시간으로 정밀지도 및 교통관제 시스템에 반영하는 것을 목표로 합니다. 이를 통해 도로 인프라 관리의 효율성을 높이고, 자율주행 및 스마트시티 구축에 필요한 최신 도로 환경 데이터를 제공합니다.

## 주요 기능

- AI기반 객체 인식 기능
- 변화 탐지 및 데이터 갱신 기능
- 정보 연동 기능
